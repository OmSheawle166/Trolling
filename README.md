# Trolling
Project Title

Fancy Moving Button Using HTML, CSS, and JavaScript

Description

This project demonstrates a fancy animated moving button using:

HTML
CSS
JavaScript

The button moves randomly across the screen whenever the user moves the mouse over it.
When clicked, it displays an alert message.

Features
Attractive gradient background
Stylish animated button
Mouse hover event handling
Random button movement
Click event alert message
Smooth transition effects
Technologies Used
